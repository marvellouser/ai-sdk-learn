# 视频网站分析测试用例

| 用例编号 | 模块 | 场景 | 输入 | 操作步骤 | 预期结果 | 优先级 |
| --- | --- | --- | --- | --- | --- | --- |
| VA-001 | 首页入口 | 首页展示视频网站分析卡片 | 无 | 打开首页 | 展示“视频网站分析”卡片，卡片链接为 `/video-analysis` | P0 |
| VA-002 | 页面导航 | 点击首页卡片进入视频分析页面 | 无 | 点击“视频网站分析”卡片 | 进入 `/video-analysis` 页面 | P0 |
| VA-003 | URL 校验 | 输入非 URL 文本 | `abc` | 点击“分析” | 展示请输入有效视频页面地址的错误提示 | P0 |
| VA-004 | URL 校验 | 输入非白名单站点 | `https://example.com/video/1` | 点击“分析” | 后端拒绝并提示当前仅支持白名单视频网站 | P0 |
| VA-005 | 安全校验 | 重定向到非白名单站点 | 白名单短链最终跳到非白名单 host | 调用 metadata 接口 | 后端拒绝最终 URL 并返回错误 | P0 |
| VA-006 | 通用解析 | 解析 Open Graph 元信息 | 包含 `og:title`、`og:description`、`og:image` 的 HTML | 调用解析函数 | 输出标题、描述和封面 | P0 |
| VA-007 | 通用解析 | 解析 JSON-LD VideoObject | 包含 `VideoObject` 的 HTML | 调用解析函数 | 输出标题、描述、封面、作者、时长、发布时间 | P0 |
| VA-008 | YouTube | 解析 YouTube 最小页面 | 包含 `ytInitialPlayerResponse` 的 HTML | 调用解析函数 | 输出 YouTube 来源、标题、封面、清晰度和字幕轨道 | P0 |
| VA-009 | bilibili | 解析 bilibili 最小页面 | 包含 `__INITIAL_STATE__` 和 `__playinfo__` 的 HTML | 调用解析函数 | 输出 bilibili 来源、标题、封面、清晰度和字幕轨道 | P0 |
| VA-010 | douyin | 解析 douyin 最小页面 | 包含公开 meta 和 hydration JSON 的 HTML | 调用解析函数 | 输出 douyin 来源、标题、封面和可用 warnings | P1 |
| VA-011 | 字幕 | 无公开字幕 | 页面不包含 captions/subtitle | 点击“分析” | 页面展示视频信息，并提示未发现公开字幕 | P0 |
| VA-012 | 字幕 | 字幕文本截断 | 字幕内容超过限制 | 调用字幕处理函数 | 输出字幕预览被截断，并生成截断 warning | P1 |
| VA-013 | AI 分析 | 有字幕时流式分析 | 已解析视频含字幕文本 | 点击“AI分析视频” | 页面流式展示视频概览、内容提纲、重点信息、传播/受众分析、可执行建议 | P0 |
| VA-014 | AI 分析 | 无字幕时流式分析 | 已解析视频无字幕文本 | 点击“AI分析视频” | AI 输出说明分析依据有限，并基于元信息分析 | P0 |
| VA-015 | 页面状态 | 分析加载状态 | 合法 URL | 点击“分析” | 按钮进入加载态，成功后展示结果并启用 AI 分析按钮 | P1 |
| VA-016 | 页面状态 | AI 流式加载状态 | 已解析视频 | 点击“AI分析视频” | 按钮进入加载态，结果区域持续更新 | P1 |
| VA-017 | 响应式 | 移动端展示 | 视口宽度小于 768px | 打开页面并解析视频 | 输入区、封面、信息卡片和 AI 结果不重叠 | P2 |
