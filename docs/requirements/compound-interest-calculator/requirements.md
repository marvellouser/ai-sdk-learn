# 复利计算器需求文档

## 背景与目标

当前首页以功能卡片作为入口，已有学习路径、自媒体文案和 AI 资讯功能。新增“复利计算器”用于在不依赖后端接口的前提下，帮助用户快速估算固定本金或定期定额投资在复利条件下的终值，并支持按目标金额反推所需期数或收益率。

目标：

- 在首页新增“复利计算器”卡片，点击进入独立页面。
- 页面全部计算在前端完成，不调用后端接口。
- 支持正向计算终值，也支持围绕目标金额做反推。
- 结果需要同时展示汇总值和每期明细，方便用户理解增长过程。

## 用户场景

- 用户已有一笔初始本金，希望知道按某个年化收益率复利后，若干年或若干月后的金额。
- 用户计划每年或每月固定投入，希望估算长期定投后的最终金额、累计本金和收益。
- 用户有一个目标金额，希望知道在固定收益率下需要投入多少期。
- 用户有一个目标金额和计划期数，希望知道需要达到多少年化收益率。

## 功能范围

### 首页入口

- 首页功能卡片区新增“复利计算器”。
- 卡片点击跳转 `/compound-interest`。
- 卡片文案需要明确这是纯前端、用于本金/定投/目标反推的计算工具。

### 页面结构

- 页面顶部展示功能标题、简短说明和“返回功能入口”按钮。
- 页面提供两类计算模式：
  - 计算终值。
  - 目标反推。
- 页面提供两类投资方式：
  - 固定初始本金。
  - 初始本金 + 每期定额投入。
- 页面提供计算期数单位：
  - 年。
  - 月。
- 定投模式提供投入时点：
  - 期末投入，默认。
  - 期初投入。

## 输入字段

### 通用字段

- 初始本金：必填，金额数字，不能小于 0。
- 期数单位：必选，年或月。
- 投资方式：必选，固定本金或定投复利。

### 计算终值

- 计算期数：必填，正整数。
- 年化收益率：必填，百分比数字。输入 `8` 表示年化 `8%`。
- 每期投入金额：定投模式必填，不能小于 0。

### 目标反推

- 目标金额：必填，金额数字，必须大于 0。
- 反推类型：
  - 输入收益率，计算所需期数。
  - 输入期数，计算所需收益率。
- 输入年化收益率计算期数时：
  - 年化收益率必填。
- 输入期数计算年化收益率时：
  - 计算期数必填，正整数。
- 每期投入金额：定投模式必填，不能小于 0。

## 输出字段

### 汇总结果

- 总本金：初始本金 + 每期投入金额 * 期数。
- 总收益：最终金额 - 总本金。
- 最终金额：计算得到的终值。

### 每期明细

每期明细至少展示：

- 期数编号。
- 累计本金。
- 当期收益。
- 期末金额。

目标反推场景中：

- 反推期数时，展示达到目标金额所需的最小整数期数，并按该期数生成明细。
- 反推收益率时，展示所需年化收益率，并按该收益率生成明细。

## 计算规则

- 收益率统一按年化收益率输入：
  - 选择“年”时，每期收益率等于年化收益率。
  - 选择“月”时，每期收益率按等效月复利率换算：`periodRate = (1 + annualRate) ^ (1 / 12) - 1`。
- 固定本金终值公式：
  - `finalAmount = principal * (1 + periodRate) ^ periods`
- 定投复利通过逐期模拟计算：
  - 期末投入：先计算当期收益，再加入当期投入。
  - 期初投入：先加入当期投入，再计算当期收益。
- 总收益：
  - `totalProfit = finalAmount - totalPrincipal`
- 目标期数反推：
  - 返回达到目标金额所需的最小整数期数。
  - 目标金额已达到时返回 0 期。
- 目标收益率反推：
  - 固定本金优先使用公式计算。
  - 定投模式使用二分法求解。

## 校验与边界

- 初始本金、每期投入、目标金额、年化收益率、期数必须是有效数字。
- 计算终值的期数必须大于 0。
- 目标反推收益率时，期数必须大于 0。
- 固定本金模式下，如果初始本金为 0 且没有投入，不能反推收益率。
- 0 收益率需要按无复利增长处理：
  - 固定本金最终金额等于初始本金。
  - 定投最终金额等于初始本金加累计投入。
- 当目标金额无法在合理范围内通过收益率求解时，页面展示明确错误提示。

## 默认值

- 计算模式：计算终值。
- 投资方式：固定本金。
- 期数单位：年。
- 投入时点：期末投入。
- 初始本金：10000。
- 每期投入：1000。
- 期数：10。
- 年化收益率：8。
- 目标金额：100000。
