# 复利计算器测试用例

| 用例编号 | 模块 | 场景 | 输入 | 操作步骤 | 预期结果 | 优先级 |
| --- | --- | --- | --- | --- | --- | --- |
| CI-001 | 首页入口 | 首页展示复利计算器卡片 | 无 | 打开首页 | 展示“复利计算器”卡片，卡片链接为 `/compound-interest` | P0 |
| CI-002 | 页面导航 | 点击首页卡片进入计算器 | 无 | 点击“复利计算器”卡片 | 进入 `/compound-interest` 页面 | P0 |
| CI-003 | 计算终值 | 固定本金按年复利 | 初始本金 10000，期数 2 年，年化收益率 10% | 选择固定本金并点击计算 | 最终金额 12100，总本金 10000，总收益 2100，明细 2 期 | P0 |
| CI-004 | 计算终值 | 固定本金按月复利 | 初始本金 10000，期数 2 月，年化收益率 12% | 选择月并点击计算 | 按等效月复利率计算，最终金额约 10190.68，明细 2 期 | P0 |
| CI-005 | 计算终值 | 定投期末投入 | 初始本金 1000，每期投入 100，期数 2 年，年化收益率 10% | 选择定投、期末投入并点击计算 | 最终金额 1420，总本金 1200，总收益 220，明细 2 期 | P0 |
| CI-006 | 计算终值 | 定投期初投入 | 初始本金 1000，每期投入 100，期数 2 年，年化收益率 10% | 选择定投、期初投入并点击计算 | 最终金额 1441，总本金 1200，总收益 241，明细 2 期 | P0 |
| CI-007 | 目标反推 | 固定本金按年化收益率反推期数 | 初始本金 10000，目标金额 12100，年化收益率 10% | 选择目标反推、计算所需期数并点击计算 | 返回需要 2 期，最终金额不低于目标金额 | P0 |
| CI-008 | 目标反推 | 定投按年化收益率反推期数 | 初始本金 1000，每期投入 100，目标金额 1420，年化收益率 10%，期末投入 | 选择目标反推、计算所需期数并点击计算 | 返回需要 2 期，展示 2 期明细 | P0 |
| CI-009 | 目标反推 | 固定本金按期数反推收益率 | 初始本金 10000，目标金额 12100，期数 2 年 | 选择目标反推、计算所需收益率并点击计算 | 返回年化收益率约 10% | P0 |
| CI-010 | 目标反推 | 定投按期数反推收益率 | 初始本金 1000，每期投入 100，目标金额 1420，期数 2 年，期末投入 | 选择目标反推、计算所需收益率并点击计算 | 返回年化收益率约 10% | P1 |
| CI-011 | 目标反推 | 月度期数反推年化收益率 | 初始本金 10000，目标金额 12100，期数 24 月 | 选择目标反推、计算所需收益率并点击计算 | 返回年化收益率约 10% | P0 |
| CI-012 | 目标反推 | 目标金额已达到 | 初始本金 10000，目标金额 9000，年化收益率 8% | 选择计算所需期数并点击计算 | 返回需要 0 期，最终金额等于初始本金 | P1 |
| CI-013 | 边界值 | 0 收益率固定本金 | 初始本金 10000，期数 10，年化收益率 0% | 点击计算 | 最终金额 10000，总收益 0 | P1 |
| CI-014 | 边界值 | 0 收益率定投 | 初始本金 1000，每期投入 100，期数 3，年化收益率 0% | 点击计算 | 最终金额 1300，总本金 1300，总收益 0 | P1 |
| CI-015 | 校验 | 计算终值期数为空 | 期数为空 | 点击计算 | 显示期数必填或请输入有效期数的错误提示 | P0 |
| CI-016 | 校验 | 初始本金为负数 | 初始本金 -1 | 点击计算 | 显示初始本金不能小于 0 的错误提示 | P0 |
| CI-017 | 校验 | 定投金额为负数 | 每期投入 -100 | 点击计算 | 显示每期投入不能小于 0 的错误提示 | P0 |
| CI-018 | 校验 | 目标金额无效 | 目标金额 0 | 点击目标反推计算 | 显示目标金额必须大于 0 的错误提示 | P0 |
| CI-019 | 交互 | 一键重置 | 修改任意字段 | 点击重置按钮 | 表单恢复默认值，结果和错误提示清空 | P2 |
| CI-020 | 展示 | 每期明细表展示 | 任意可计算输入 | 点击计算 | 明细表包含期数、累计本金、当期收益、期末金额 | P1 |
| CI-021 | 响应式 | 移动端展示 | 视口宽度小于 768px | 打开页面并计算 | 表单、结果卡片和明细表不重叠，表格可横向滚动 | P2 |
